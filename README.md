# Ippo Vitrine

Site vitrine React (Vite) pour l'application mobile Ippo.

## Contenu

- Présentation de l'application
- Galerie de captures d'écran (placeholders à remplacer)
- Liens de téléchargement App Store / Google Play
- Page `Règles de confidentialité` accessible via `#/privacy`

## Démarrage

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## À personnaliser avant publication

- Remplacer les liens App Store / Google Play dans `src/App.jsx`
- Remplacer les captures dans `public/screenshots/`
- Remplacer l'adresse e-mail de contact dans `src/content/privacyPolicy.js`
